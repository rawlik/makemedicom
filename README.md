# makemedicom

## Installation
First install pipx: https://pypa.github.io/pipx/

```bash
pipx install git+https://github.com/rawlik/makemedicom.git
```

## Usage
```bash
makemedicom myfile.h5
```

## Development
```bash
git clone git@github.com:rawlik/makemedicom.git
cd makemedicom
pipx install --force .
```
